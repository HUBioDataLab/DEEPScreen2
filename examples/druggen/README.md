# DrugGEN Use-Case: Screening Generated Molecules

This example demonstrates how to use DEEPScreen2 to screen de novo generated molecules (from DrugGEN) against a specific target (CHEMBL4282).

## Abstract
**Target-specific de novo design of drug candidate molecules with graph-transformer-based generative adversarial networks**

Discovering novel drug candidate molecules is a fundamental step in drug development. Generative deep learning models can sample new molecular structures from learned probability distributions; however, their practical use in drug discovery hinges on generating compounds tailored to a specific target molecule. Here we introduce DrugGEN, an end-to-end generative system for the de novo design of drug candidate molecules that interact with a selected protein. The proposed method represents molecules as graphs and processes them using a generative adversarial network that comprises graph transformer layers. Trained on large datasets of drug-like compounds and target-specific bioactive molecules, DrugGEN designed candidate inhibitors for AKT1, a kinase crucial in many cancers. Docking and molecular dynamics simulations suggest that the generated compounds effectively bind to AKT1, and attention maps provide insights into the modelâ€™s reasoning. Furthermore, selected de novo molecules were synthesized and shown to inhibit AKT1 at low micromolar concentrations in the context of in vitro enzymatic assays. These results demonstrate the potential of DrugGEN for designing target-specific molecules. Using the open-access DrugGEN codebase, researchers can retrain the model for other druggable proteins, provided a dataset of known bioactive molecules is available.

## Goal
To filter and identify promising drug candidates from a large library of generative model outputs that are predicted to be active against CHEMBL4282.

## Data Source
*   **Activity Data**: `training_files/CHEMBL4282/activity_data.csv` - Training data for the target.
*   **Prediction Data**: `prediction_file/DrugGEN_generated_molecules.csv` - Molecules generated by DrugGEN.

## Usage

Run all commands from the repository root directory (`DEEPScreen2/`).

### 1. Training

Train the model on the target data.

```bash
python main_training.py \
    --target_id CHEMBL4282 \
    --model CNNModel1 \
    --training_dir examples/druggen/training_files \
    --pchembl_threshold 6.0 \
    --en druggen_experiment \
    --project_name deepscreen_druggen
```

### 2. Prediction

Screen the DrugGEN generated molecules using the trained model.

```bash
python predict_deepscreen.py \
    --target_id DrugGEN_screening \
    --model_path examples/druggen/model/CHEMBL4282_best_val-CHEMBL4282-CNNModel1-512-256-0.001-256-0.3-20-deepscreen_scaffold_balanced_lr0.001_drop0.3_bs256-state_dict.pth \
    --smiles_file examples/druggen/prediction_file/DrugGEN_generated_molecules.csv
```

## Outputs

*   **Predictions**: The output file will contain the predicted activity for each generated molecule, allowing you to select the best candidates for further verification.
